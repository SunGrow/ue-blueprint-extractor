Pipeline Studio — Final Design

Visual node-based LLM workflow orchestrator. 3 documents approved through 3 review rounds.

What it is

A desktop app where you build LLM workflows visually — drag nodes onto a canvas, connect them with typed wires, run everything with one click. Each node can use a different model and CLI tool. The engine runs headlessly in Rust. An embedded MCP server makes the tool programmable by LLMs themselves.

Key capabilities

4 LLM backends
Claude Code CLI, Gemini CLI, Codex CLI (native with full ecosystems) + OpenRouter API (200+ models). Auto-detects installed CLIs; grays out missing ones with install instructions.
Typed data flow
13 data types with color-coded wires. Type mismatches blocked at wire-draw time. Struct outputs can be split into individual fields and routed to different nodes.
Safe by default
Wire-time type checking. Build-time graph validation. Per-node cost budgets. Process sandboxing. Every error port must be handled or explicitly ignored. Opt-out like Rust's unsafe.
Functions & loops
Select nodes, extract to reusable function. Functions nest and share across projects. For/ForEach/While/Break loops with per-iteration debugging.
Graph editing
Multi-select, copy-paste (within and across graphs), extract-to-function, undo/redo. Standard node editor UX.
LLM-editable
Built-in MCP server with 13 tools. LLMs can create nodes, wire them, validate, execute. Pipelines are programmable infrastructure.
Incremental execution
Input-signature caching. Changed one node? Only it and its downstream re-run. Unchanged steps skipped.
Performance-conscious
256KB buffered child I/O (10-50x syscall reduction). Zero-copy bytes::Bytes for streaming. simd-json parsing. 16ms event batching for UI.
Protocol-first
JSON-RPC 2.0 (same as LSP and MCP). Evaluated against gRPC, Cap'n Proto, FlatBuffers, MessagePack — JSON-RPC wins on Tauri compat, MCP sharing, and LLM readability.
